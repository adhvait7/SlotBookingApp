Slot Booking System

Effortlessly schedule appointments and classes for gyms and institutions.

About This Project

SlotBookingApp is a dynamic scheduling platform built with Flask and MongoDB Atlas. It provides role-based dashboards for staff and students, real-time availability updates, and a secure login system to streamline appointment management.

Upcoming Features

Feedback Module

Collect user feedback directly and store it in the database for continuous improvement.

Notification System

Real-time email and SMS alerts for bookings, cancellations, and reminders.

Admin Controls

Advanced admin panel to manage users, slots, and analytics.
